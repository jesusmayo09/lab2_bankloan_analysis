# 💳 Bank Loan Analysis

## 📌 Descripción

Este proyecto analiza datos de préstamos bancarios utilizando Python y Pandas.

El notebook incluye:

- Limpieza de datos
- Detección de valores atípicos
- Análisis exploratorio (EDA)
- Correlaciones entre variables
- Identificación de perfiles con mayor riesgo de incumplimiento (`Default`)

---

## 🧰 Tecnologías

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Principales hallazgos

- Los usuarios jóvenes presentan mayor probabilidad de incumplimiento.
- Se detectaron outliers en variables como `income` y `age`.
- Algunas variables financieras muestran correlación positiva fuerte.
